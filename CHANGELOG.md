## 0.0.1 (2026-03-11)

### Features

- Heating curve calculator with real-time visualization
- PID control configuration with deadband support
- ESPHome YAML generator with Mustache templates
- URL parameter sharing for configurations
- Presets system with LocalStorage persistence
- Dark/light theme support
- GitHub Pages deployment

### Refactoring

- Monorepo with core/web packages
- Migrate to shadcn/ui and Tailwind CSS
- Extract UI concerns from core to web
- Unified sidebar with industrial control room aesthetic

### Documentation

- Reflect shadcn/tailwind migration
- Remove empty constants.ts from codebase
- Improve YAML generation and fix documentation
